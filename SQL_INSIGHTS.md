# SQL Learning Insights

## ORDER BY & LIMIT

### What I Learned
- ORDER BY sorts data.
- ASC is ascending.
- DESC is descending.
- LIMIT restricts the number of rows returned.

### Common Mistakes
- Using LIMIT before ORDER BY.
- Forgetting DESC when finding top values.

---

## GROUP BY & HAVING

### What I Learned
- GROUP BY groups rows.
- COUNT(), SUM(), AVG() work with groups.
- HAVING filters grouped data.

### Common Mistakes
- Using aggregate functions in WHERE.
- Selecting columns not included in GROUP BY.

---

## JOINS

### What I Learned
- INNER JOIN returns matching records.
- LEFT JOIN returns all records from the left table.

### Common Mistakes
- Missing ON condition.
